# Sales Analysis

## Project Objective

Analyzing this dataset allows us to uncover seasonal sales trends, highlight regional performance, compare sales representatives, and better understand customer behavior

## Task

1. Sales Performance
Which region sells the most?
Which sales rep brings in the highest revenue?
Which products/categories are most/least profitable?
2. Seasonal/Time Analysis
Detect seasonal patterns (monthly/quarterly sales trends).
Check peak months for sales per region/product.
3. Customer Insights
Compare new vs returning customers in terms of revenue.
Find customer type preferences (e.g., which category do new customers buy more?).
5. Discount Effectiveness
Does giving a discount increase quantity sold or revenue?
6. Channel & Payment Analysis
Compare online vs retail performance.
Which payment method is most popular per region/customer type?
7. Sales Rep & Region Synergy
Measure average sales per rep.
Identify best performing region-rep pairs.
